# 🚀 Briefing Flow

**Briefing Flow** é uma aplicação web desenvolvida para equipes de marketing e agências criativas com o objetivo de eliminar retrabalho causado por briefings desalinhados ou incompletos.

A plataforma atua como um **sistema de entrada e validação de demandas**, garantindo que apenas briefings claros, estruturados e completos cheguem ao time criativo.

---

## 🎯 Objetivo do Projeto

- Reduzir retrabalho causado por briefings mal elaborados  
- Padronizar o processo de criação de briefing  
- Aumentar a produtividade do time  
- Melhorar a qualidade das entregas  
- Acelerar o fluxo de produção  

---

## 🧠 Problema Resolvido

Atualmente, o fluxo em muitas agências segue este padrão:

Cliente solicita → Atendimento interpreta → Briefing incompleto → Execução com suposições → Revisões → Retrabalho

O **Briefing Flow** atua diretamente nesse gargalo, criando um **filtro de qualidade antes da execução**.

---

## ⚙️ Funcionalidades Principais

- 📋 Formulário de briefing estruturado (multi-etapas)  
- ✅ Validação automática de informações  
- 📊 Score de qualidade do briefing  
- 🎯 Definição clara de objetivo, público e direção criativa  
- 🖼️ Upload obrigatório de referências  
- 🧠 Geração de direção criativa automatizada  
- 📌 Dashboard com status dos briefings
